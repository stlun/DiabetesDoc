DiabetesDoc ist eine Software zur Verwaltung von Diabetes-Daten. Sie ermöglicht neben der dauerhaften Speicherung der Daten aus Dateien und dem SmartPix-Gerät auch die Ausgabe eines Tagebuch im PDF-Format.

Funktionen

Die Software wird über die Menü-Leiste, die Button-Spalte auf der linken Seite, sowie die Tagesübersicht-Fenster gesteuert.

Menü-Leiste

Die Menü-Leiste enthält Schaltflächen zum Einlesen, zur Ausgabe, sowie für die Optionen.

Einlesen

Das Programm liest die XML-Daten von der angegebenen Quelle ein und speichert sie. Als Quelle stehen von SmartPix (Strg + R) und aus Datei... (Strg + F) zur Verfügung.
Bei ersterer Option (von SmartPix) greift das Programm auf den unter Einstellungen angegebenen Pfad für das SmartPix-Gerät zu und liest die Daten ein, sowie speichert den Report unter dem Pfad reports.
Bei zweiterer Option (aus Datei...) fragt das Programm vom Nutzer eine XML-Datei ab, die eingelesen werden soll. Für einen alten Report finden Sie die XML-Dateien im Verzeichnis reports/"DATUM"/xml/.

Ausgabe

Als Ausgabeformate steht als PDF... (Strg + P) zur Verfügung. Nach Eingabe des gewünschten Zeitraums für das Tagebuch wird die PDF-Datei im Verzeichnis pdf unter dem Namen startdatum-enddatum.pdf gespeichert (Datumsformat jeweils JJJJ-MM-TT).

Optionen

Unter Optionen stehen die Einstellungen, allgemeine Informationen über DiabetesDoc und diese Hilfe zur Verfügung.
Die Bedeutung der einzelnen Einstellungen ergibt sich wie folgt:

SmartPix-Directory:Der Pfad an dem sich das SmartPix-Gerät befindet.
Username: Der Nutzername für die Ausgabe (kann auch leer bleiben).
Geburtstag: Der Geburtstag des Nutzers für die Ausgabe (kann auch leer bleiben).

Button-Spalte

In der Button-Spalte werden alle eigelesenen Tage angezeigt. Durch Klick auf einen Tag öffnet sich für diesen ein Tagesübersicht-Fenster.

Tagesübersicht-Fenster

Im Tagesübersicht-Fenster werden für den durch die Button-Leiste ausgewählten Tag die XML-Datei, eine Tabellenübersicht und eine Blutzuckerübersicht in Bildform angezeigt. Im oberen Text-Bereich können die XML-Daten angepasst werden (siehe dazu XML-Dateien bearbeiten). Das Menü hat folgende Schaltflächen und Bedeutungen:

Datei | Speichern (Strg + S): Speichert die (u.U. bearbeitete) XML-Datei, wie sie im Fenster angezeigt wird.
Datei | Datei Schließen (Strg + F4): Schließt die XML-Datei und das Tagesübersicht-Fenster.
Bearbeiten | Rückgängig (Strg + Z): Macht die letzte Bearbeitung der XML-Daten im Fenster rückgängig.
Bearbeiten | Wiederholen (Strg + Y): Wiederholt die letzte Bearbeitung der XML-Daten im Fenster.
Bearbeiten | Ausschneiden (Strg + X): Schneidet den ausgewählten Bereich der XML-Daten aus.
Bearbeiten | Kopieren (Strg + C): Kopiert den ausgewählten Bereich der XML-Daten aus.
Bearbeiten | Einfügen (Strg + V): Fügt den Text aus der Zwischenablage im ausgewählten Bereich der XML-Daten ein.
Bearbeiten | BG hinzufügen: Fügt ein BG-Tag im ausgewählten Bereich der XML-Daten hinzu.
Bearbeiten | BOLUS hinzufügen: Fügt ein BOLUS-Tag im ausgewählten Bereich der XML-Daten hinzu.
Bearbeiten | BASAL hinzufügen: Fügt ein BASAL-Tag im ausgewählten Bereich der XML-Daten hinzu.
Bearbeiten | EVENT hinzufügen: Fügt ein EVENT-Tag im ausgewählten Bereich der XML-Daten hinzu.
