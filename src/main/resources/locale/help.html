DiabetesDoc is a software for management of Diabetes-data. Aside permanent saving data from files or a SmartPix-Device, you can also create a diary in PDF format.

Functions

You control the Software by the menu-bar, the button-column on the left side, and the Day-Overview-Frames.

Menu-bar

The menu-bar contains buttons to Read, for Output and for Options.

Read

The program reads the XML-data from the given source and saves it. Possible sources are from SmartPix (Ctrl + R) and from file... (Ctrl + F).
The first Option (from SmartPix) uses the path given by the Settings for the SmartPix-device as source, reads the data and saves the report into the directory reports.
The second Option (from file...) asks the user for an XML-file, which is read. A Report's XML-files are in the directory reports/"DATE"/xml.

Output

The program asks for a period for the diary and saves it as PDF... (Ctrl + P) into the directory pdf with the name startDate-endDate.pdf (format for dates: YYYY-MM-DD).

Options

At Options there are the Settings, general information About DiabetesDoc and this Help.
The meaning of the several settings is the following:

SmartPix-Directory:The Path of the SmartPix-Device.
Username:The user's name for the output (can stay empty).
Birthday:The user's birthday for the output (can stay empty).

Button-column

In the button-column all read days are shown. By Clicking on a date, an Day-Overview-Frame opens for it.

Day-Overview-Frame

In the Day-Overview-Frame the date's XML-data chosen by the button-column, a table-overview and a blood-glucose-overview as image are shown. In the text-area on the top, the XML-data can be edited (see Edit XML-files). The menu has the following buttons an meaning:

File | Save (Ctrl + S):Saves the (possibly edited) XML-file, as it is displayed in the Frame.
File | Save As:Saves the (possibly edited) XML-file, but asks the user, for a file-name.
File | Close File (Ctrl + F4):Closes the XML-file and the Day-Overview-Frame.
Edit | Undo (Ctrl + Z):Undoes the last edit of the XML-data in the Frame.
Edit | Redo (Ctrl + Y):Redoes the last edit of the XML-data in the Frame.
Edit | Cut (Ctrl + X):Cuts the selected part of the XML-data.
Edit | Copy (Ctrl + C):Copies the selected part of the XML-data.
Edit | Paste (Ctrl + V):Pastes the text from the clip-board to the selected part of the XML-data.
Edit | add BG:Adds a BG-tag to the selected part of the XML-data.
Edit | add BOLUS:Adds a BOLUS-Tag to the selected part of the XML-data.
Edit | add BASAL:Adds a BASAL-Tag to the selected part of the XML-data.
Edit | add EVENT:Adds an EVENT-Tag to the selected part of the XML-data.
